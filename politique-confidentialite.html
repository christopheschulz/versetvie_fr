Politique de confidentialité

Le site VersetVie respecte votre vie privée.

Collecte de données sur le site web

  • Nous ne collectons aucune donnée personnelle sur ce site.
  • Nous n'utilisons aucun cookie de suivi ou d'analyse.
  • Nous ne partageons aucune information avec des tiers.
  • Si vous nous contactez par email, votre adresse ne sera utilisée que pour répondre à votre demande.

Collecte de données dans l'application mobile

  • L'application VersetVie utilise Mixpanel pour collecter des données d'usage anonymes.
  • Ces données nous permettent d'améliorer l'application et l'expérience utilisateur.
  • Toutes les données collectées sont entièrement anonymes et ne permettent pas de vous identifier personnellement.
  • Aucune donnée personnelle (nom, email, localisation précise) n'est collectée ou stockée.
  • Vous avez le choix d'accepter ou de refuser cette collecte lors du premier lancement de l'application (onboarding).
  • Vous pouvez également modifier ce choix à tout moment dans l'application via le menu "Politiques de confidentialité".

Pour toute question, nous contacter.
